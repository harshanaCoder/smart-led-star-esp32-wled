# ⭐ Smart IoT-Based LED Star (ESP32 + WLED)

## 📌 Overview
This project is a smart decorative lighting system built using ESP32 and WS2812 addressable LEDs. It uses WLED firmware to enable real-time wireless control and dynamic lighting effects.

## 🚀 Features
- WiFi-based control using ESP32
- Dynamic lighting effects via WLED
- Individually addressable WS2812 LEDs
- Custom star-shaped design with flowing LED tail
- Real-time brightness and color control

## 🛠️ Hardware Used
- ESP32
- WS2812 LED Strip
- 5V Power Supply
- Connecting wires and mounting board

## 🔌 Wiring Overview
- Data Pin → ESP32 GPIO (e.g., D5)
- VCC → 5V Power Supply
- GND → Common Ground

## ⚙️ Software
- WLED firmware for LED control
- Web interface for real-time configuration
