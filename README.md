# ⭐ Smart IoT-Based LED Star (ESP32 + WLED)

## 📌 Overview
This project is a smart IoT-based decorative lighting system developed using an ESP32 microcontroller and WS2812 addressable LEDs.  
The system is powered by WLED firmware, enabling real-time wireless control, dynamic lighting effects, and customizable animations through a web interface.

The project focuses on combining **embedded systems, IoT control, and hardware–software integration** to create an interactive lighting solution.

---

## 🚀 Key Features
- WiFi-based LED control using ESP32  
- Real-time brightness, color, and effect customization  
- Individually addressable WS2812 LED control  
- Custom star-shaped LED design with a flowing light tail  
- Web-based interface using WLED firmware  
- Scalable design for future smart lighting applications  

---

## 🛠️ Hardware Components
- ESP32 Development Board  
- WS2812 Addressable LED Strip  
- 5V High-Current Power Supply  
- Connecting Wires  
- Mounting Board / Structure  

---

## 🔌 Wiring Overview
| Component | Connection |
|--------|------------|
| LED Data Pin | ESP32 GPIO (example: D5 / GPIO5) |
| LED VCC | 5V External Power Supply |
| LED GND | Common Ground (ESP32 + Power Supply) |

---

## ⚙️ Software Used
- WLED Firmware for LED control  
- Web-based interface for real-time configuration  
- WiFi communication using ESP32
